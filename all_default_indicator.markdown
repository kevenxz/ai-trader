移动平均线 (Moving Average, MA)

类型：简单移动平均线 (SMA) 或指数移动平均线 (EMA)。EMA对近期价格更敏感。

相对强弱指数 (Relative Strength Index, RSI)

描述：衡量价格动量和超买/超卖条件。

参数：

周期：通常为14（也可调整為9用于更敏感的信号，或25用于更平滑的信号）。

移动平均收敛散度 (MACD)

描述：显示趋势变化、动量和方向。

参数：

快线EMA周期：通常为12。

慢线EMA周期：通常为26。

信号线周期（SMA）：通常为9。

布林带 (Bollinger Bands, BB)

描述：测量波动性和识别超买/超卖水平。

参数：

周期：通常为20。

标准差倍数：通常为2（上轨和下轨各基于2倍标准差）。

随机振荡器 (Stochastic Oscillator)

描述：比较收盘价与价格范围，识别反转点。

参数：

%K周期：通常为14。

%D周期（%K的SMA）：通常为3。

减速：通常为3（平滑参数）。

平均真实范围 (Average True Range, ATR)

描述：测量市场波动性，用于设置止损和止盈。

参数：

周期：通常为14。

商品通道指数 (Commodity Channel Index, CCI)

描述：识别超买/超卖条件和趋势强度。

参数：

周期：通常为20。

威廉姆斯 %R (Williams %R)

描述：类似随机振荡器，但刻度反向，用于识别超买/超卖。

参数：

周期：通常为14。

动量 (Momentum)

描述：测量价格变化速度，确认趋势强度。

参数：

周期：通常为10（比较当前收盘价与N周期前的收盘价）。

顺势指标 (Ichimoku Cloud)

描述：综合趋势、支撑/阻力和动量指标。

参数：

转换线周期 (Tenkan-sen)：通常为9。

基准线周期 (Kijun-sen)：通常为26。

滞后跨度周期 (Chikou Span)：通常为26。

先行跨度周期 (Senkou Span A和B)：通常为52和26。

抛物线转向指标 (Parabolic SAR)

描述：提供止损和反转信号，适用于趋势市场。

参数：

步长：通常为0.02（每次价格移动时加速因子的增量）。

最大步长：通常为0.2（加速因子的上限）。

成交量加权平均价 (Volume Weighted Average Price, VWAP)

描述：基于成交量加权的平均价，常用于日内交易。

参数：

通常基于日内数据（如1小时或4小时），无固定周期，但计算时使用当日所有K线数据。

资金流量指数 (Money Flow Index, MFI)

描述：结合价格和成交量的RSI变体，测量资金流入流出。

参数：

周期：通常为14。

平衡成交量 (On-Balance Volume, OBV)

描述：基于成交量的累积指标，确认趋势强度。

参数：无周期，基于累计成交量（每个K线的成交量根据价格变化正负累加）。

累积/分布线 (Accumulation/Distribution Line, A/D)

描述：基于价格和成交量的资金流指标。

参数：无周期，基于每个K线的资金流累加。

蔡金资金流量 (Chaikin Money Flow, CMF)

描述：测量资金流入流出强度。

参数：

周期：通常为20。

标准差 (Standard Deviation)

描述：测量价格波动性。

参数：

周期：通常为20。

平均方向指数 (Average Directional Index, ADX)

描述：衡量趋势强度，不指示方向。

参数：

周期：通常为14。

波动率指标 (Volatility Indicators)

描述：如历史波动率，基于价格标准差计算。

参数：

周期：通常为20（计算收益率的标准差）。

斐波那契回撤 (Fibonacci Retracement)

描述：基于数学比例识别支撑和阻力位。

参数：无固定参数，但基于价格高低点（例如，从波段高点到低点绘制，常用回撤水平为23.6%、38.2%、50%、61.8%、78.6%）。
